## Key Findings - Project 4

1. Average shipping time is 3.96 days
2. Discount strongly reduces profit - confirmed 
   by both correlation (-0.22) and regression 
   coefficient (-233.46)
3. Sales positively correlates with profit (0.48) 
   but only explains 27% of profit variation
4. Quantity has minimal impact on profit (0.07)

## Models Built
- Linear regression on Sales, Quantity, Discount 
  predicting Profit
- R² Score: 0.27 (train), tested on holdout data
- Conclusion: Discount level is the strongest 
  controllable factor affecting profitability
